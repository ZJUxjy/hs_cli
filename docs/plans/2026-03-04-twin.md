## Task 2: 在 GameEngine 中处理双生

**Files:**
- Modify: `src/game/GameEngine.js`

**Step 1: 修改 summonMinion 方法处理双生**

在调用 battlecry 后添加:

```javascript
// 触发战吼
if (minion.battlecry) {
  const CardEffect = require('./CardEffect');
  const cardEffect = new CardEffect(this);

  // 第一次触发
  cardEffect.executeBattlecry(card, { player, target: player, card });

  // 双生：效果触发两次
  if (minion.twin) {
    Logger.info(`${minion.name} 的双生效果，触发第二次战吼`);
    cardEffect.executeBattlecry(card, { player, target: player, card });
  }
}
```

**Step 2: 在召唤时设置双生标记**

在存储其他效果的位置:

```javascript
// 存储战吼、亡语、激励、复古、法术爆发和双生
minion.battlecry = card.effect?.battlecry || null;
minion.deathrattle = card.effect?.deathrattle || null;
minion.inspire = card.effect?.inspire || null;
minion.reborn = card.effect?.reborn || false;
minion.twin = card.effect?.twin || false;
```

**Step 3: 同时处理亡语双生**

```javascript
// 触发亡语
deadMinions.forEach(minion => {
  if (minion.deathrattle) {
    const CardEffect = require('./CardEffect');
    const cardEffect = new CardEffect(this);

    // 第一次触发
    cardEffect.executeDeathrattle(minion, { player, target: opponent, card: minion });

    // 双生：亡语也触发两次
    if (minion.twin) {
      Logger.info(`${minion.name} 的双生效果，触发第二次亡语`);
      cardEffect.executeDeathrattle(minion, { player, target: opponent, card: minion });
    }
  }
});
```

**Step 4: 测试验证**

Run: `node -e "const ge = require('./src/game/GameEngine'); console.log('Twin logic added');"`
Expected: 无错误输出
