# i18n 多语言支持设计

## 概述

为 Hearthstone WebUI 项目添加国际化（i18n）支持，支持中文和英文两种语言，允许用户在游戏界面随时切换语言。

## 技术方案

- **库**: react-i18next + i18next
- **配置**: 自动检测浏览器语言， 切换时自动保存到 localStorage
- **切换器**: 顶部导航栏下拉菜单
- **语言文件**:
  - `src/locales/zh.json` (中文)
  - `src/locales/en.json` (英文)
